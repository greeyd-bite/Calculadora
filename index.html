Inicio

Esta página fue creada con el fin de presentar nuestro proyecto de la calculadora de Fundamentos de Programación.La presente investigación tiene como propósito abordar el desarrollo y la implementación de una herramienta digital básica, como una calculadora web, en el contexto del aprendizaje de fundamentos de programación. Esta iniciativa busca fortalecer las competencias técnicas de los estudiantes mediante la práctica y la creación de soluciones funcionales, aplicadas a problemas cotidianos.

Calculadora





Sobre la página

Es un proyecto con el fin de representar lo aprendido en clase. Diseñar y desarrollar una calculadora web funcional utilizando HTML, CSS y JavaScript, como ejercicio práctico en el proceso de formación en programación básica.

Enfoque metodológico

  • Se utilizó una metodología de tipo exploratorio-práctico, basada en la aplicación directa de lenguajes de programación web. La estructura del proyecto se dividió en fases: diseño de interfaz, codificación de funciones aritméticas y pruebas de funcionamiento. Este enfoque permite afianzar el aprendizaje a través de la experiencia.

Programación

Esta sección resalta cómo aplicamos los conocimientos adquiridos en clase de Fundamentos de Programación. Utilizamos HTML para estructurar la página, CSS para diseñarla visualmente y JavaScript para darle funcionalidad. La creación de esta calculadora es un ejemplo práctico de cómo la programación nos permite resolver problemas reales mediante el uso de lenguajes de codificación.
